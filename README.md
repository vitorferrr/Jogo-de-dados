# Jogo-de-dados
Jogo de dados para 2 jogadores. Usando React e NextJS, a ideia é montar um jogo que consiste em uma batalha entre 2 jogadores que dura 5 rodadas. Basicamente o jogador que tirar o maior valor na soma dos dois dados arremessados ganha a rodada, ao concluir 5 rodadas vencidas ele garante a vitória. Caso o valor seja igual, é dado empate.
